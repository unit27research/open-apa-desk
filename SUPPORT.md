# Support

Open APA Desk support is handled by Unit27 Research.

For help, bug reports, or review questions, contact:

- Email: josh@unit27research.com
- GitHub issues: https://github.com/unit27research/open-apa-desk/issues

Please do not include private paper content, student records, passwords, API
tokens, private Google Doc URLs, or unpublished school information in public
GitHub issues.

Open APA Desk is not an official APA product and does not replace instructor,
institution, or APA manual requirements.
